# Google Play 데이터 보안 설문 초안

이 문서는 Play Console의 데이터 보안 섹션을 작성할 때 참고하기 위한 내부 초안입니다. 실제 제출 전 앱의 최종 기능과 다시 대조해야 합니다.

## 앱 기본 방향

- 앱 이름: 행웃아웃행
- 개발자 표시 이름: DailySmile Studio
- 문의 이메일: dailysmile.help@gmail.com
- 개인 웃음 사진: 기기 내부 로컬 저장
- 친구에게 보내는 웃긴 사진 및 선택적으로 전송한 반응 사진: Firebase Storage 저장
- 친구, 초대 코드, 랭킹, 웃기기 결과: Firebase Firestore 저장
- Firebase 익명 인증 UID: 친구 기능 문서 식별과 접근 제한에 사용
- 푸시 토큰: 사용자가 친구 활동 알림을 켠 경우 알림 전송에 사용
- 광고: 현재 사용하지 않음
- 결제: 현재 사용하지 않음

## 수집 또는 처리하는 데이터

### 사진 및 동영상

- 처리 여부: 예
- 용도: 앱 기능
- 설명: 카메라 촬영, 갤러리에서 사용자가 선택한 이미지, 웃음 판별, 웃음 기록 저장, 친구에게 웃긴 사진 전송, 사용자가 선택한 반응 사진 전송. 서버에 전송되는 친구 기능 사진은 업로드 후 7일이 지나면 자동 삭제 대상으로 처리됨.
- 서버 전송 여부: 개인 웃음 기록 사진은 서버 전송 없음. 친구 기능 사진은 Firebase Storage에 저장될 수 있음.
- 공유 여부: 사용자가 공유 또는 친구 전송을 선택한 경우에만 공유

### 개인 식별 정보

- 처리 여부: 제한적으로 예
- 항목: 닉네임, 초대 코드, Firebase 익명 사용자 ID
- 용도: 친구 추가, 친구 랭킹, 웃긴 사진 도전 기능
- 서버 저장 여부: Firebase Firestore 저장

### 앱 활동 또는 앱 정보

- 처리 여부: 제한적으로 예
- 항목: 촬영 날짜, 웃음 판별 결과, 웃김 정도, 점수, 친구 도전 결과
- 용도: 기록 표시, 점수 계산, 랭킹 표시
- 서버 저장 여부: 개인 웃음 기록은 로컬 저장. 친구 기능 결과는 Firebase Firestore 저장.

### 기기 또는 기타 ID

- 처리 여부: 제한적으로 예
- 항목: 푸시 알림 토큰
- 용도: 친구 요청, 웃기기 도전, 웃기기 결과 알림 전송
- 서버 저장 여부: 사용자가 친구 활동 알림을 켠 경우 Firebase Firestore에 저장
- 선택 여부: 사용자가 설정에서 알림을 끄면 현재 기기의 토큰을 서버에서 삭제

### 위치, 연락처, 결제 정보, 건강 정보, 마이크 음성

- 처리 여부: 아니오

## 보안 및 전송

- Firebase 저장 데이터는 Firebase 보안 규칙으로 접근을 제한합니다.
- 앱은 HTTPS 기반 Firebase 서비스를 사용합니다.
- 사용자의 개인 웃음 기록은 기본적으로 기기 내부에 보관합니다.

## 사용자가 선택할 수 있는 항목

- 카메라 권한 허용 여부
- 알림 권한 허용 여부
- 랭킹 점수 공개 여부
- 웃긴 사진 반응 사진 전송 여부
- 기록 삭제

## Play Console에서 주의할 항목

- 개인정보처리방침 URL은 공개 접속 가능한 웹 URL이어야 합니다.
- PDF가 아닌 웹페이지 형태가 권장됩니다.
- 앱 내부 또는 설정 화면에도 개인정보처리방침 접근 경로를 두는 것이 좋습니다.
- 사진/동영상 광범위 접근 권한은 피해야 합니다. 현재 배포 APK에서는 READ_EXTERNAL_STORAGE, WRITE_EXTERNAL_STORAGE를 제거했습니다.
