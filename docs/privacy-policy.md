# 개인정보처리방침

시행일: 2026년 6월 2일

DailySmile Studio는 앱 "행웃아웃행"을 운영하며, 사용자의 개인정보와 사진 데이터를 안전하게 다루기 위해 이 개인정보처리방침을 제공합니다.

## 1. 수집하거나 처리하는 정보

앱은 다음 정보를 기능 제공을 위해 처리할 수 있습니다.

- 카메라 촬영 이미지: 오늘의 웃음 사진 촬영, 웃음 판별, 친구에게 보내는 웃긴 사진 또는 반응 사진 전송에 사용됩니다.
- 웃음 기록 정보: 촬영 날짜, 웃음 판별 결과, 웃김 정도, 얼굴 감지 수, 연속 촬영 점수, 총 촬영 수가 저장됩니다.
- 친구 기능 정보: 닉네임, 초대 코드, 친구 요청 상태, 랭킹 공개 여부, 친구에게 보낸 웃긴 사진 도전과 반응 결과가 저장됩니다.
- 알림 설정 정보: 사용자가 설정한 알림 시간 또는 시간 범위가 저장됩니다.

앱은 위치 정보, 연락처, 결제 정보, 건강 정보, 마이크 음성 데이터를 수집하지 않습니다.

## 2. 사진 데이터 처리 방식

사용자가 매일 촬영하는 개인 웃음 사진은 기본적으로 사용자 기기 내부에만 저장됩니다. 이 사진은 Firebase 등 외부 서버로 자동 업로드되지 않습니다.

다만 친구 기능을 사용할 때 다음 사진은 Firebase Storage에 저장될 수 있습니다.

- 친구에게 보내는 웃긴 사진
- 친구가 웃긴 사진을 확인한 뒤 사용자가 전송하기로 선택한 반응 사진

반응 사진 전송은 사용자가 직접 토글을 켠 경우에만 진행됩니다.

## 3. 얼굴 및 웃음 판별

앱은 사용자의 사진에서 얼굴과 웃음 정도를 판별하기 위해 기기 내 얼굴 인식 기능을 사용합니다. 판별 결과는 웃음 기록, 도전 성공 여부, 점수 계산에 사용됩니다.

개인 웃음 사진의 얼굴 분석 결과는 사용자의 기기 내부 기록에 저장됩니다. 친구 기능에서 웃기기 성공 여부가 발생한 경우에는 도전 결과와 점수 정보가 Firebase에 저장될 수 있습니다.

## 4. 정보 이용 목적

DailySmile Studio는 정보를 다음 목적으로만 사용합니다.

- 하루 한 장 웃음 사진 기록 제공
- 달력 및 최근 기록 표시
- Combo Smile, Max Smile, 총 웃음 사진 수 계산
- 알림 설정 및 알림 발송
- 친구 초대, 친구 랭킹, 웃긴 사진 도전 기능 제공
- GIF 생성 및 공유 기능 제공
- 앱 안정성 유지와 오류 대응

## 5. 정보 공유

DailySmile Studio는 사용자의 개인정보를 광고 목적이나 제3자 판매 목적으로 공유하지 않습니다.

친구 기능에서 필요한 일부 데이터는 Firebase 서비스를 통해 저장 및 처리됩니다. 이 데이터는 앱 기능 제공을 위한 목적으로만 사용됩니다.

사용자가 GIF 공유, 사진 공유 등 공유 기능을 직접 실행하는 경우, 사용자가 선택한 외부 앱 또는 서비스로 파일이 전달될 수 있습니다. 이 경우 해당 외부 서비스의 개인정보 처리방침이 적용될 수 있습니다.

## 6. 보관 및 삭제

Firebase에 저장되는 친구 기능 데이터는 기능 제공을 위해 보관됩니다. 사용자가 삭제를 요청하는 경우 DailySmile Studio는 합리적인 기간 내에 삭제 요청을 처리합니다.
